Nuevo plugin for Videojs framework version 7.x

Video.js player file used in `examples` directory is "video.min.js", located in "videojs" directory.
This is video.js minimized script, version 7.21.4 released 2023-03-15.
Full version of videojs is located in the same directory.


You may also use any CDN hosted video.js version above 7.2. Please note that delivered by Nuevodevel.com video.js scripts are slightly modified for better experience when the user drag and move playing progress bar.


Upload "videojs7" directoryto your server.
Navigate to "videojs7/examples/index.html" for multiple HTML examples of videojs v8 and Nuevo v11, covering most popular videojs and nuevo plugin usage.
